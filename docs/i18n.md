# Internacionalización (i18n) — Inglés y Español

## Contexto

El spa opera en **Florida, Estados Unidos**, con clientela hispanohablante e inglesa.
El sistema debe estar disponible en **ambos idiomas** con cambio instantáneo.

---

## Stack de i18n

| Librería | Uso |
|---|---|
| `next-intl` | Traducción de la UI, rutas y mensajes |

```bash
npm install next-intl
```

```
src/
└── messages/
    ├── en.json     ← textos en inglés (idioma base)

```

---

## Selector de idioma en la UI

- **Ubicación**: esquina superior derecha en todas las páginas
- **Presentación**: botones o toggle `EN | ES`
- **Persistencia**: guardar preferencia en `localStorage` y en `clients.preferred_language`
- **Default**: inglés (`en`) — por ser negocio en EE.UU.

```tsx
// Ejemplo de selector
<div className="flex gap-2">
  <button
    onClick={() => setLocale('en')}
    className={locale === 'en' ? 'font-bold text-amber-600' : 'text-gray-400'}
  >
    EN
  </button>
  <span className="text-gray-300">|</span>
  <button
    onClick={() => setLocale('es')}
    className={locale === 'es' ? 'font-bold text-amber-600' : 'text-gray-400'}
  >
    ES
  </button>
</div>
```

## Reglas para Claude Code

1. **Nunca hardcodear strings de UI** — usar siempre las keys del sistema i18n
2. **Todos los nombres de campos del formulario** deben mostrarse en el idioma activo
3. **Las fechas** se formatean según el locale:
   - `en`: `April 7, 2026`
   - `es`: `7 de abril de 2026`
4. **Los números de teléfono** siempre en formato US: `(407) 388-4928`
5. **Los precios** siempre en USD: `$80` o `USD 80` según contexto
6. **El idioma del código** sigue siendo inglés — solo cambia la UI visible al usuario

## Pantalla de check-in en tablet — es la más usada

La pantalla `/scan` debe detectar el idioma preferido del cliente desde su perfil
y mostrarlo en ese idioma. Si no tiene preferencia guardada, usar el idioma activo del sistema.

```typescript
// Al cargar el resultado del check-in:
const displayLocale = checkinResult.client.preferred_language ?? currentLocale
```
